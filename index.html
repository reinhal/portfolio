JusKidin is a web all-in-one tool for parents. The user creates profiles for each child by entering a name and a birthdate. Then, through a custom Google API, parents can get child developmental information for each child. Additionally, the user can create drawers. These drawers can be used to organize images by topic, event or some other custom title. Parents have a lot to manage and track. The vision of this app is to give parents a real tool to help. Also, children attend many events, create art and school projects. By digitizing thses items and memories, parents will be able to keep them all.
